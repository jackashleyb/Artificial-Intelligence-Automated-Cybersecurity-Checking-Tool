# Artificial-Intelligence-Automated-Cybersecurity-Checking-Tool

this project helps to check the risks to LLMs and Ai systems by automating a Threat Modelling & Guardrail System 

I am currently building an automation tool that connects a local LLM to map data flow and analyse risks like prompt injection, model jailbreaking and data leakage referencing the OWASP top 10 for LLMs and MITRE ATLAS frameworks.
